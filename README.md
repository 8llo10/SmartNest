# SmartNest
SmartNest is a modern smart home web dashboard that allows users to monitor and control devices, rooms, security systems, energy usage, and family access from a single interface. Designed for large homes and villas, it provides a clean, responsive, and user-friendly experience with smart automation features and real-time home management.

## DevOps workspace

This repository uses the `smartnest/` app as the Jenkins CI/CD target.

- Jenkins pipeline: `smartnest/Jenkinsfile`
- Docker file: `smartnest/Dockerfile`
- Demo runbook: `smartnest/docs/GHLO_DEMO_RUNBOOK.md`
- Validation log: `smartnest/docs/GHLO_PIPELINE_RESULTS.md`
